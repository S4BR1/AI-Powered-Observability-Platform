# AI-Powered-Observability-Platform

Plateforme DevOps intelligente permettant de :

1. Analyser automatiquement un dépôt GitHub
2. Détecter la stack applicative
3. Générer Dockerfile + manifestes Kubernetes
4. Déployer sur AKS
5. Ajouter l'observabilité OpenTelemetry
6. Diagnostiquer automatiquement les incidents

## Architecture du projet

Voir le dossier `infrastructure` pour AKS + ACR.
Voir le dossier `platform` pour l'intelligence du projet.
Voir le dossier `k8s` pour les manifests Kubernetes.
